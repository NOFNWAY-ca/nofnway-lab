VAPID Key Generator

Generates a pair of cryptographic keys for Shhhhh... push notifications. Run this once. Keep the private key secret — it never leaves your browser.

What to do with these:
1. Go to Cloudflare → Pages → nofnway-lab → Settings → Environment Variables
2. Add VAPID_PUBLIC_KEY with the first value
3. Add VAPID_PRIVATE_JWK with the second value (mark it as secret)
4. Save and redeploy

Don't run this again unless you also re-subscribe all devices — changing keys breaks existing subscriptions.
